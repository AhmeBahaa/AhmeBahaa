### 🚀 About Me

Electronics and Communications Engineer with a passion for bringing hardware to life. I love mixing embedded systems, IoT, and cloud technologies to build impactful tech solutions that actually help people.

🔭 &nbsp;I'm currently working on **Prepping my  bionic arm for the IEEE YESIST12 Grand Finale and refining a non-invasive glucose monitoring system  with a research focus on each one of them .**  
🌱 &nbsp;I'm currently learning **Everything related to modern communications engineering, from advanced network infrastructure to telecommunications technologies.**  
👯 &nbsp;I'm looking to collaborate on **Open-source hardware, assistive technologies, and any cool cloud IoT project.**  
🤔 &nbsp;I'm looking for help with **just sharing good resources on network optimization.**  
💬 &nbsp;Ask me about **Microcontrollers ,robotics and automation engineering, network engineering, or building full-stack IoT setups.**  
😄 &nbsp;Pronouns: **he**  
⚡ &nbsp;Fun fact: **I can spend days debugging a code issue only to realize it was a loose jumper wire all along!**
